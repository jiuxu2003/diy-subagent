# CC-Switch 风格解剖（视觉验收参照）

来源：farion1231/cc-switch 仓库 `assets/screenshots/main-zh.png`、`add-zh.png`（2026-07-22 下载实测，本地副本 `/tmp/ccswitch-main-zh.png`、`/tmp/ccswitch-add-zh.png`）。其前端栈为 React + TailwindCSS 3.4 + shadcn/ui，与本项目（Tailwind 4 + Radix + CVA）同源可平移。

## 主界面（main-zh.png）

- 无侧栏。整窗浅灰背景，overlay 红绿灯悬浮左上。
- 顶栏单行：左 = 应用名（蓝色、粗体、约 28px）+ 齿轮图标；中右 = 应用分段切换器（白底圆角容器内三个「图标+文字」段，选中段白底加阴影浮起）；右 = 一组图标按钮（圆角矩形描边容器）+ 橙色圆形「+」按钮（约 44px，白色加号）。
- 内容区 = 单列卡片流，每卡一条供应商：
  - 卡片：白底、圆角约 16px、1px 极浅描边 + 软阴影（hover 略抬升）、垂直内边距大（整卡高约 120px）、卡间距约 20px。
  - 卡内左起：六点拖拽手柄（灰）→ 图标瓷砖（约 56px，白底圆角 14px、浅描边，内嵌品牌图标）→ 名称（约 20px 粗体）+ 第二行蓝色链接。
  - 激活卡追加：名称右侧绿色软胶囊「当前使用」（浅绿底、深绿字、rounded-full）；卡右侧「刚刚 ⟳」+「已使用/剩余」数字（关键数字着色）。

## 添加页（add-zh.png）

- 顶部：白色圆角方块返回箭头按钮 + 页标题「添加 Claude Code 供应商」（约 22px 粗体），下方发丝线分隔。
- 正文为一个大圆角容器：
  - 「预设供应商」小节标题（约 15px，普通字重）。
  - 预设胶囊组：rounded-full 灰底胶囊（内边距约 12×24px），可多行 wrap；选中项变实色（截图中红色底白字）；部分胶囊右上角有星标角标。
  - 胶囊组下一行提示：「💡 只需填写 API Key，请求地址已预设」。
  - 居中大号品牌图标（约 96px 圆角瓷砖）。
  - 表单：标签在上（约 15px），输入框大圆角（约 12px）、高约 48px、浅描边；两列布局（供应商名称 / 备注）。
- 右下角操作对：「取消」灰底圆角按钮 + 「＋ 添加」蓝色主按钮（rounded-xl）。

## 映射到本项目的量化基线

| 维度 | CC-Switch 观察值 | 本项目采用 |
| --- | --- | --- |
| 卡片圆角 | ≈16px | 16px（列表卡）/ 12px（输入框）/ full（胶囊） |
| 卡片描边+阴影 | 1px 浅描边 + 软阴影 | `--border` + 升级版 `--shadow-card`（多层软阴影） |
| 图标瓷砖 | 56px 圆角 14px | 名称首字母 monogram 瓷砖（无品牌图标数据） |
| 主色 | 蓝（链接/标题/主按钮）+ 橙「+」 | 系统蓝 #007AFF/#0A84FF + 品牌紫蓝 #6c74f6「+」（D3） |
| 状态表达 | 软胶囊（浅色底+深色字） | Pill 原语取代 StatusDot |
| 正文字号 | 约 15-16px | 保持 14px 基准，标题层级放大（用户偏 14-15px） |
| 分段切换器 | 白底容器 + 选中段浮起 | 同构，承载 Claude Code / Codex / Cursor |

## 与 07-21「反 AI 味」结论的相容性

CC-Switch 的卡片是扁平单层数据行：无嵌套卡、无营销眉标、无内部术语。07-21 真正反对的是「卡片套卡片 + 胶囊堆叠 + 术语外漏」，因此本次采用胶囊/卡片不构成回退；「术语不进 UI」结论继续生效。
